# Refinery
Проект направлен на создание открытых датасетов по НПЗ и нефтепроводам России.

## ПРОБЛЕМЫ
Нехватка открытых данных по НПЗ России

## РЕШЕНИЕ
Создание Датасетов и интерактивной карты НПЗ и нефтепроводов России.
